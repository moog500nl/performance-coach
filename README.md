# Zwift Coach

A personal performance coaching workspace powered by Claude. Forked from CrankAddict/section-11

This repository contains training data, workout plans, and performance reports generated by an AI coaching assistant tailored for cycling and endurance sports.

## Contents

* **Training data** — synced activity files and performance metrics
* **Workout plans** — structured sessions based on current fitness
* **Reports** — periodic performance summaries and trend analysis
* **Protocols** — coaching guidelines and methodology

## About

This workspace is used to track and analyse training progress, generate personalised workout recommendations, and monitor long-term fitness development.
